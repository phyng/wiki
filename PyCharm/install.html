Install PyCharm in Ubuntu and fix fonts

Ubuntu install PyCharm

http://cheparev.com/pycharm-installation-on-ubuntu-13-10/

#install jdk, a PPA from thirtpart, download from Oracle & auto-install-config
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java7-installer

#unpack
tar xfz pycharm-*.tar.gz
cp

#run & config
/opt/pycharm/bin/pycharm.sh

#chown to update
chown :<user> -R /opt/pycharm

Which JDK?

Both Oracle JDK and OpenJDK have fonts error.

Fix fonts error

  • http://www.webupd8.org/2013/06/install-openjdk-patched-with-font-fixes.html
  • https://launchpad.net/~no1wantdthisname/+archive/ubuntu/openjdk-fontfix

# Install
sudo add-apt-repository ppa:no1wantdthisname/openjdk-fontfix
sudo apt-get update
sudo apt-get install openjdk-7-jdk

# Uninstall
sudo apt-get install ppa-purge
sudo ppa-purge ppa:no1wantdthisname/openjdk-fontfix

# Change JDK
sudo update-alternatives --config java

# Config JDK
sudo gedit /etc/profile 
## Add this:
export _JAVA_OPTIONS="-Dawt.useSystemAAFontSettings=lcd \
                      -Dsun.java2d.xrender=true"
