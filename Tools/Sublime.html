快捷键/快捷操作

* Shift + F11                       免打扰模式
* Shift + 鼠标右键                  列选择模式
* Ctrl + L                          选择单行
* Ctrl + P                          文件切换
* Ctrl + D                          加选单词或者块选中
* Ctrl + G                          行定位(:)
* Ctrl + J                          行合并 
* Ctrl + M                          光标在括号内跳转
* Ctrl + `                          命令面板
* Ctrl + /                          插入注释
* Ctrl + ;                          查找文件中的变量名、属性名(#)
* Ctrl + F2                         设置书签，且以F2切换书签
* Ctrl + F3                         定位所选文本的相同文本
* Ctrl + K + B                      开启/关闭侧边栏
* Ctrl + K + K                      删除光标后的文本
* Ctrl + Shift + D                  向下复制行
* Ctrl + Shift + K                  删除当前行
* Ctrl + Shift + T                  恢复关闭的标签页
* Ctrl + 鼠标点击                    多处编辑
* Ctrl + Shift + 方向键上下          移动该行
* Alt + .                           闭合当前标签
* Alt + F3                          一次性选择全部的相同文本进行同时编辑
* Alt + Shift + 1 - 7               水平分屏
* Alt + Shift + 8 , 9               垂直分屏

安装插件

  • SideBarEnhancements 侧边栏增强
  • SFTP SFTP
  • Markdown Editing 带高亮和自动完成的Markdown编辑
  • Markdown Preview Markdown预览

配置选项

"default_line_ending": "unix",  //换行风格
"trim_trailing_white_space_on_save": true,//保存时自动删除行尾空格
"ensure_newline_at_eof_on_save": true,//保存时自动在文件末尾加EOF
"tab_size": 4,//tab对应空格
"translate_tabs_to_spaces": true,//tab自动转换成空格

参考文档

  • http://www.bluesdream.com/blog/sublime-text-snippets-function.html

示例

<snippet>
<content>
<![CDATA[
---
title: "${1:title}"
layout: page
date: 2014-${2:07-16} 00:00
---
]]>
</content>
<!-- Optional: Set a tabTrigger to define how to trigger the snippet -->
<tabTrigger>mdt</tabTrigger>
<!-- Optional: Set a scope to limit where the snippet will trigger -->
<scope>text.html.markdown</scope>
</snippet>
